Здесь нужно разместить файл logo.png с логотипом BARLAU.KZ размером 1024x1024 пикселей.

Для создания иконок и сплэш-экранов различных размеров используйте инструмент cordova-res:

```bash
npm install -g cordova-res
cordova-res ios --skip-config --copy
```

Требования к логотипу:
- Формат: PNG
- Размер: 1024x1024 пикселей
- Прозрачный фон (если нужно)
- Без скругленных углов (скругление будет добавлено iOS автоматически)
